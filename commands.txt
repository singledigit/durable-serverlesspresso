# Durable Functions Commands Reference

## Local Development Commands

### Local Invocation
sam local invoke CoffeeOrdersFunction --event events/coffee-order-test.json --durable-execution-name test-001 -n locals.json

### Execution Management (Local)
# Get execution details
sam local execution get $EXID

# View execution history (table format)
sam local execution history $EXID

# View execution history (JSON format)
sam local execution history $EXID --format json

# Stop a running execution
sam local execution stop $EXID

### Callback Management (Local)
# Send success callback
sam local callback succeed $CBID --result '{"action": "ACCEPT", "baristaId": "barista-123"}'

# Send heartbeat
sam local callback heartbeat $CBID

# Send failure callback
sam local callback fail $CBID --error '{"message": "Rejected"}'

## Remote Commands

### Remote async invocation
sam remote invoke 'arn:aws:lambda:us-east-1:458865386569:function:sev2-CoffeeOrdersFunction-go7T447B7EMr:$LATEST' --event-file ./order.json

### Remote sync invocation
sam remote invoke CoffeeOrdersFunction --event-file events/order.json

### Execution Management (Remote)
# Get execution details
sam remote execution get $EXID --profile private --region us-east-1

# View execution history
sam remote execution history $EXID --profile private --region us-east-1

# View execution history (JSON format)
sam remote execution history $EXID --format json --profile private --region us-east-1

# Stop a running execution
sam remote execution stop $EXID --profile private --region us-east-1
